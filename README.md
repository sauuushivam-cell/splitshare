# 💸 SplitShare

SplitShare is a simple and responsive web application that helps users track and split shared expenses with friends. It makes managing group expenses easy for trips, meals, rent, and other shared activities.

## ✨ Features

- ➕ Add shared expenses
- 👥 Manage friends
- 💰 Track who paid and who owes
- 🎨 Clean and responsive user interface
- 🗑️ Reset all data with one click
